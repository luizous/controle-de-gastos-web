- Projeto ASP.NET core web (Web aplication mvc)
- Instalar EntityFrameworkCore.Sql pelo Nuget
- Botão direito na pasta onde está a solution -> git bash here -> 

Configuração iniciais do git:
- git config --global user.name "luizous"
- git config --global user.email "luizaruivoms@gmail.com"

Criar um repositório local(navegar até a pasta do projeto):
- git init 

Verificar o status do repositório:
- git status

Adicionar os arquivos para o monitoramento do git:
- git add * (o asterisco indica todos os arquivos da pasta)

Criar um commit no repositório:
- git commit -m "Commit inicial do projeto"

Configurar para onde o repositório será enviado (isto é feito apenas uma vez!):
- git remote add origin https://github.com/luizous/controle-de-gastos.git

Enviar o commit para o repositório remoto:
- git push -u origin master

Remover credenciais:
- Ir no buscador do windows e pesquisar por: Credenciais

Mudar de branch:
- git checkout luiza

*                             INSTRUÇÕES SOBRE O EF CORE                          *
* * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * 
1 - Instalar o EF Core no projeto
	- Install-Package Microsoft.EntityFrameworkCore.SqlServer -Version:2.1.11

2 - Criar as classes de modelo

3 - Classe de contexto
	- Herdar do DbContext
	- Criar o construtor
	- Criar o DbSet de cada classe de modelo

4 - Configurar o serviço do banco de dados 
na classe statup.cs

5 - Criar a string de conexão no arquivo 
appsettings.json

6 - Adicionar a migração
	- Add-Migration NomeMigracao

7 - Atualizar o banco
	- Update-Database -verbose
